データ管理

PCを変える、ブラウザのデータを消す、スマホでも使いたい場合は、定期的にJSON保存してください。

GitHub Pagesに置くと、URLを知っている人はページを開けます。ただし、この設計ではタスクデータは各ブラウザ内に保存され、GitHubに自動送信されません。
